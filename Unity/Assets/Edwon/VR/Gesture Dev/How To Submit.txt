
To Prep and Submit to the Asset Store

1. Prepare Examples
- if examples have updated since the last time a package was made
	- select all examples in "Data"
	- select all examples in "Examples" except Getting Started
	- File > Export Package 
	- export it back to "Examples folder"
	- move all examples in "Data" to "Gesture Dev" folder
	- move all examples in "Examples" to "Gesture Dev" folder except Getting Started

2. Prepare Integrations
- if integrations have changed since the last time a package was made
	- select a top level integration folder, e.g. "Playmaker"
	- File > Export Package
	- export it back to "Integrations"
	- move "Playmaker" folder to "Gesture Dev" folder

3. Delete all Test Data
- delete everything inisde the "Data" folder
- but be sure to leave "Data" folder itself intact

4. Export Package or upload to Asset Store Submission
