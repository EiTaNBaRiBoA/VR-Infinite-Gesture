
VR Infinite Gesture
Tutorials ReadMe

To do the tutorial,

1. import the "Tutorials" package inside this folder
2. open the "Getting Started" scene inside the Getting Started folder
3. once the scene opens, hit Play
4. if the scene is in VR, restart the scene (the first half of the tutorail is non-VR)

The first half of the tutorial will help you setup VR. 
The second half of the tutorial is in VR.

You'll need to stop the scene to do certain steps, so watch out for the yellow caution signs.
You can only go watch videos and go between steps while the scene is running!

Navigate between steps using the "NEXT" and "BACK" buttons or the arrow keys on the keyboard.
