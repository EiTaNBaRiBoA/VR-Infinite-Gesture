
VR Infinite Gesture
Getting Started Tutorial ReadMe

SteamVR Tutorial:
1) Create New Project
2) Import VRInfinite Gesture package
3) Import SteamVR package https://www.assetstore.unity3d.com/en/#!/content/32647
4) Create New Scene
5) Drag in [Camera Rig] prefab from SteamVR > Prefabs
6) Attach VR Gesture Rig component and click Auto Setup
7) Change VR Infinite Gesture settings. Go Tools > VR Infinite Gesture > Click Settings Tab
8) Change VR Type to SteamVR
9) Toggle "Show VRUI" to On
10) Press Play
11) Create a new neural net on the UI
12) Create a new gesture: either one or two-handed in the UI
13) Hold trigger/s and create many versions of the gesture
14) Click the back arrow on the UI
15) Process the neural net
16) Click the Detect mode
17) Try your new gesture
